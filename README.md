# VedaAi-assessment-creator-
VedaAI is a comprehensive, web-based teacher workspace designed to streamline the academic lifecycle—from assessment generation and rubric building to class management.

Core Functionality

AI-Powered Exam Generation: The platform uses an integrated Gemini API pipeline to generate structured, curriculum-aligned exam papers based on user-provided topics, grade levels, and specific pedagogical constraints.

Assessment Designer: A multi-step wizard allows educators to configure section types (MCQ, Short Answer, Long Answer/Essay) and difficulty levels, with real-time feedback during the generation process.

Rubric Toolkit: A specialized tool that generates standardized evaluation criteria (Excellent to Novice levels) based on specific subjects and learning outcomes.

Library & Archive: A repository system where teachers can store, retrieve, and modify past assessments, maintaining a history of their work.

User Interface & Experience

Professional Dashboard: Built with Tailwind CSS and the "Plus Jakarta Sans" font, it provides a clean, distraction-free environment for teachers to track statistics, such as active classes and AI tokens used.

Print-Ready Optimization: The application features a dedicated @media print layer that automatically strips away the web UI (sidebars, buttons, headers) when printing or saving as PDF, ensuring professional, exam-ready documents.

Interactive Simulation: The interface includes a "Queue Monitor" that visualizes the backend process (API calls, Redis caching, job queuing), providing transparency during the AI generation process.

Target Audience

The platform is specifically built for K-12 and secondary school educators who need to create assessments that comply with standardized grading (e.g., CBSE standards), requiring high levels of precision and academic integrity.
